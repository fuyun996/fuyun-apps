# 中国象棋游戏

这是一个纯前端单页中国象棋小游戏，直接打开 `index.html` 就可以开始双人对弈、人机对战或载入残局。

## 已实现功能

- 标准 9 x 10 中国象棋棋盘
- 红黑双方轮流走子
- 双人对弈与人机对战
- 多档 AI 难度
- 标准棋子走法判定
- 将军检测
- 自动过滤导致己方继续被将军的非法着法
- 将帅照面禁手
- 将死 / 无合法着法判定
- 内置残局库
- 悔棋
- 重新开始
- 着法记录
- 棋谱文本导出与 JSON 下载
- 门户返回入口

## 文件说明

- `index.html`：页面结构
- `styles.css`：页面样式和棋盘布局
- `app.js`：棋规和交互逻辑
- `xiangqi-doc.md`：说明文档

## 使用方式

1. 打开 `index.html`
2. 在右侧选择双人或人机模式，并按需切换残局与 AI 难度
3. 点击当前行棋方的棋子
4. 选择高亮显示的合法落点完成走子
5. 通过“悔棋”撤销上一步，或通过“重新开始”重开对局
6. 在导出区域复制文本棋谱，或下载 JSON 棋谱文件

## 当前版本说明

- 人机模式下默认玩家执红，AI 执黑
- 残局库可一键切换到几组预设局面
- 棋谱导出会包含模式、残局名称、结果和完整着法记录
- 规则判定已覆盖常见基础走法与将军约束
