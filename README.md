# Fuyun Apps

一个以纯前端小游戏为主的合集仓库，当前包含统一门户首页、围棋小游戏和中国象棋小游戏。

## 项目结构

- `portal/`
  小游戏门户首页，作为多个小游戏的统一入口
- `weiqi/`
  围棋小游戏，支持多棋盘尺寸、人机对战、复盘、SGF 导入导出等能力
- `xiangqi/`
  中国象棋小游戏，支持双人对弈、人机对战、残局库和棋谱导出
- `deploy-nginx.sh`
  静态文件部署脚本
- `DEPLOY.md`
  Nginx 部署说明

## 当前已接入游戏

- 围棋
- 中国象棋

## 门户能力

- 统一首页入口
- 游戏卡片展示
- 已上线 / 待上线状态区分
- 棋类与休闲分类筛选

## 中国象棋能力

- 标准走子规则与将军判定
- 双人对弈
- 人机对战
- 多档 AI 难度
- 残局库切换
- 棋谱文本导出
- JSON 棋谱下载

## 围棋能力

- 9 路 / 13 路 / 19 路棋盘
- 双人对弈与人机对战
- 多档 AI 棋力
- 中国规则 / 日本规则
- 自动数目
- SGF 导入导出
- 棋谱复盘

## 本地使用

直接用浏览器打开下面任一页面即可：

- `portal/index.html`
- `weiqi/index.html`
- `xiangqi/index.html`

## Nginx 部署

项目已按静态站点方式整理，可参考：

- `DEPLOY.md`

也可以直接执行：

```bash
bash ./deploy-nginx.sh
```
